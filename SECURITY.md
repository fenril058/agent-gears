# Security Policy

## Reporting a vulnerability

If you find a vulnerability in the skills / agents / plugins, `install.sh`, the Nix flake, or `mdidx` (the Go CLI) in this repository, please report it.

Use a private channel instead of a public issue.

- Preferred: GitHub [Private vulnerability reporting](https://github.com/fenril058/agent-gears/security/advisories/new) ("Report a vulnerability" on the Security tab).
- Alternative: email fenril.nh@gmail.com.

Where possible, include reproduction steps, the impact, and the affected files.

## Handling

This is a personally maintained repository.

No SLA is guaranteed, but reports are reviewed and reasonable issues are fixed as time permits.

## Out of scope

- The user's local environment configuration, or issues in Claude Code / Codex themselves.
- Problems caused by a user's own modifications to the skills distributed here.
